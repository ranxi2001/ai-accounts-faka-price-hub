## 项目定位

AI 账号、Plus CDK、共享订阅、API 额度等商品的价格长期很混乱。同一种类型的账号，在不同发卡平台、不同代理商那里可能卖 30、40、60，甚至更高；但追到提货渠道后会发现，很多商品其实来自同一个上游，只是中间代理的加价不同。

这个项目不卖账号，也不给任何平台背书，而是把分散在发卡站、中转站、代理商、社群里的价格和反馈整理到一起，帮用户少花时间比价，少踩坑。

本项目覆盖两类平台：

| 类型 | 说明 | 计价方式 |
|------|------|----------|
| **发卡平台** | 销售 AI 账号、CDK、订阅等 | 固定价格 (CNY) |
| **API 中转站** | 提供模型 API 转发服务 | 按 token 计价 (¥/百万token) |

**解决的问题：**

> - 发卡网、中转站、代理商、社群信息分散，比价耗时
> - 同源商品被多层加价，用户很难判断真实成本
> - 商家售后水平参差不齐，翻车后维权困难
> - 价格和供货渠道波动频繁，旧信息很快失效
> - 低价、永久、包稳等宣传难以分辨真假

## 推荐与避雷

如果你有靠谱信源，欢迎提交。所谓靠谱，不只是价格低，还包括渠道稳定、发货清楚、售后响应及时、出问题时愿意补号或退款。

被坑过的也欢迎分享经历。真实的翻车记录比空泛的推荐更有价值。

**提交内容建议包含：**

- 商品类型（ChatGPT Plus CDK、共享号、独享号、API 额度等）
- 购买价格、购买时间和有效期
- 发货方式、提货渠道或疑似上游
- 实际可用时长、是否被封、是否补号
- 售后响应速度和处理结果

## 购买指南

**如何选择发卡网：**

1. **优先选择自动发货平台** — 避免人工等待，降低跑路风险
2. **核查有无售后保障** — 正规平台应承诺账号被封后补号或退款
3. **查看历史口碑** — 搜索平台名 + "翻车"/"被骗"，了解真实情况
4. **小额试单** — 首次购买建议先买最便宜的商品测试发货流程
5. **保留截图凭证** — 下单页、账号信息页均需截图，方便售后维权

**常见坑点：**

- 过低价格要警惕 — 市场价 65 的东西卖 15，99% 是问题账号
- 谨慎"永久账号" — 几乎不存在永久可用的共享 AI 账号
- 认准 HTTPS — 不要在 HTTP 站点输入任何支付信息
- 避开私聊交易 — 只在有担保机制的正规发卡平台购买

## 自动抓取工具

本项目自带价格抓取脚本，使用 Playwright 无头浏览器自动访问各平台提取商品数据。

**快速开始：**

```bash
# 安装依赖（首次）
bash scripts/refresh.sh --install

# 一键抓取 + 生成表格
bash scripts/refresh.sh

# 抓取并更新 README
bash scripts/refresh.sh --update-readme
```

**高级用法：**

```bash
python3 scripts/scrape_prices.py --type faka       # 只抓发卡平台
python3 scripts/scrape_prices.py --type relay       # 只抓 API 中转站
python3 scripts/scrape_prices.py --site vuvuva      # 只抓指定站点
python3 scripts/scrape_prices.py --list             # 列出所有站点
```

**目录结构：**

```
ai-accounts-faka-price-hub/
├── README.md                          # 项目主页
├── scripts/
│   ├── sites.json                     # 平台配置（URL、类型）
│   ├── scrape_prices.py               # Playwright 抓取脚本
│   ├── generate_table.py              # Markdown 表格生成
│   ├── refresh.sh                     # 一键刷新
│   └── requirements.txt               # Python 依赖
├── data/
│   ├── prices.json                    # 抓取原始数据
│   └── prices_table.md                # 生成的 Markdown 表格
├── .claude/skills/fetch-prices/       # Claude Code 技能
├── assets/
│   └── banner.png
└── LICENSE
```
