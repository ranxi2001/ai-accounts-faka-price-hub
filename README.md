## 项目定位

AI 账号、Plus CDK、共享订阅、API 额度等商品的价格长期很混乱。同一种类型的账号，在不同发卡平台、不同代理商那里可能卖 30、40、60，甚至更高；但追到提货渠道后会发现，很多商品其实来自同一个上游，只是中间代理的加价不同，账号本质没有明显差别。

这个项目要做的不是卖账号，也不是给任何平台背书，而是把这些分散在发卡站、中转站、代理商、社群里的价格和反馈整理到一起，帮助用户少花时间反复横跳比价，也尽量少踩"同源高价""售后失联""虚假库存""低价翻车"这类坑。

本项目覆盖两类平台：

- **发卡平台**：销售 AI 账号、CDK、订阅等固定价格商品
- **API 中转站**：提供模型 API 转发服务，按 token 计价

**解决的问题：**
- 发卡网、中转站、代理商、社群信息分散，比价耗时
- 同源商品被多层加价，用户很难判断真实成本
- 商家售后水平参差不齐，翻车后维权困难
- 价格和供货渠道波动频繁，旧信息很快失效
- 低价、永久、包稳等宣传难以分辨真假

## 推荐与避雷

如果你有靠谱信源，欢迎提交到这里。所谓靠谱，不只是价格低，还包括渠道稳定、发货清楚、售后响应及时、出现问题时愿意补号或退款。

如果你被坑过，也欢迎在本仓库分享经历。真实的翻车记录比空泛的推荐更有价值，可以帮助后来的人避开同一个坑。

提交内容建议包含：

- 商品类型：如 ChatGPT Plus CDK、共享号、独享号、API 额度等
- 购买价格、购买时间和有效期
- 发货方式、提货渠道或疑似上游
- 实际可用时长、是否被封、是否补号
- 售后响应速度和处理结果

## 后续观察

目前从第三方发卡渠道购买部分 AI 商品，确实可能比官方订阅便宜一些。例如低价 Plus CDK 如果能稳定使用一段时间，对只想短期体验的用户还有性价比。

但这类渠道本身不稳定：上游涨价、账号风控、售后缩水、平台跑路都会快速拉低性价比。一旦第三方价格接近官方订阅，或者可用时长明显缩短，直接选择官方订阅、合规团队方案，或可靠的多人分摊方案，可能反而更省心。

本项目会持续记录这些变化，但所有信息都只作为参考，不构成购买建议。

## 自动抓取工具

本项目自带价格抓取脚本，使用 Playwright 无头浏览器自动访问各平台提取商品数据。

### 快速开始

```bash
# 安装依赖（首次）
bash scripts/refresh.sh --install

# 一键抓取 + 生成表格
bash scripts/refresh.sh

# 抓取并更新 README
bash scripts/refresh.sh --update-readme
```

### 高级用法

```bash
# 只抓发卡平台
python3 scripts/scrape_prices.py --type faka

# 只抓 API 中转站
python3 scripts/scrape_prices.py --type relay

# 只抓指定站点
python3 scripts/scrape_prices.py --site vuvuva dragoncode

# 列出所有站点
python3 scripts/scrape_prices.py --list
```

## 购买指南

### 如何选择发卡网？

1. **优先选择自动发货平台** — 避免人工等待，降低跑路风险
2. **核查有无售后保障** — 正规平台应承诺账号被封后补号或退款
3. **查看历史口碑** — 搜索平台名 + "翻车"/"被骗"，了解真实情况
4. **小额试单** — 首次购买建议先买最便宜的商品测试发货流程
5. **保留截图凭证** — 下单页、账号信息页均需截图，方便售后维权

### 常见坑点提醒

- 过低价格要警惕 — 市场价 65 的东西卖 15，99% 是问题账号
- 谨慎"永久账号" — 几乎不存在永久可用的共享 AI 账号
- 认准 HTTPS — 不要在 HTTP 站点输入任何支付信息
- 避开私聊交易 — 只在有担保机制的正规发卡平台购买
